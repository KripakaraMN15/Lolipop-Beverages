# **App Name**: OliScape

## Core Features:

- Cinematic Parallax Hero: Full-screen hero section featuring scroll-controlled WebP image sequences as dynamic backgrounds, creating an immersive, cinematic effect.
- Dynamic Variant Display: Displays and dynamically updates drink name, subtitle, description, and visual theme based on the currently selected drink variant.
- Variant Navigation System: Intuitive right-side navigation panel for cycling through multiple drink variants (PREV/NEXT) and indicating the current flavor index (e.g., 01, 02).
- Themed Styling & Mode Toggle: Applies variant-specific theme colors to accents and CTA elements, with support for a dark/light mode toggle (defaulting to dark mode only as requested).
- Optimized Media Loading: Implements full-screen pre-loading with progress indicators for initial WebP sequence load and smaller indicators for variant switching, ensuring smooth transitions.
- External Variant Data Management: Structures and manages drink variant details (name, description, theme, WebP sequence paths, frame count) from an external data source or configuration.
- AI-Driven Copy Enhancement Tool: A tool to assist content creators in generating concise, engaging marketing copy for new drink descriptions, ingredient benefits, or social media blurbs based on input flavor profiles.

## Style Guidelines:

- Background color: Deep charcoal grey (`#18191C`, hsl(240, 5%, 10%)) providing a dramatic, cinematic canvas. Chosen for its sophisticated depth, aligning with premium branding and dark mode preference, and offering ample contrast for text and accent elements.
- Primary color: Luminous, subtle cool white (`#F1F2F5`, hsl(220, 10%, 95%)). This serves as the brand accent, creating high contrast for headlines, prominent text, and interactive elements, representing elegance and modernity against the dark background.
- Accent color: Muted, sophisticated grey-green (`#89A4AE`, hsl(190, 15%, 60%)). Used for secondary highlights, subtle indicators, and UI elements to add depth and a touch of organic modernity, complementing the cool monochrome primary palette.
- Headlines: 'Playfair' (serif, geometric) for impactful, large text elements like drink names and section titles, chosen for its elegant, high-contrast, and high-end feel, resonating with a premium brand aesthetic.
- Body Text: 'Inter' (sans-serif, grotesque) for descriptions, paragraphs, and general content, ensuring excellent legibility, a modern appearance, and neutrality that allows the visual effects to shine.
- Minimalist, monochrome icons for social media and navigation (PREV/NEXT arrows). Designs should be flat and geometric to maintain a clean, modern aesthetic that blends seamlessly with the dark, premium theme.
- Full-screen hero section with an asymmetrical content block on the left (text), a clean center for motion focus, and a vertical variant navigation on the right. Standard marketing sections flow below, adhering to a dark, minimalist, and visually cohesive structure.
- Smooth, scroll-triggered playback for WebP image sequences creates a cinematic, interactive parallax effect. Text changes upon variant selection should employ subtle fade-in/fade-out animations for a polished, seamless user experience.
